<h1>Palindrome Permutation</h1>

Given a string `s`, return `true` _if a permutation of the string could form a palindrome and_ `false` _otherwise._

<hr>

__Example 1:__
```
Input: s = "code"
Output: false
```
__Example 2:__
```
Input: s = "aab"
Output: true
```
__Example 3:__
```
Input: s = "carerac"
Output: true
```

__Constraints:__

- `1 <= s.length <= 5000`
- `s` consists of only lowercase English letters.
