<h1>Contains Duplicate</h1>

Given an integer array `nums`, return `true` if any value appears __at least twice__ in the array, and return `false` if every element is distinct.

<hr>

__Example 1:__
```
Input: nums = [1,2,3,1]
Output: true
Example 2:
```

__Example 2:__
```
Input: nums = [1,2,3,4]
Output: false
```

__Example 3:__
```
Input: nums = [1,1,1,3,3,4,3,2,4,2]
Output: true
```

__Constrains__
- `1 <= nums.length <= 105`
- `-109 <= nums[i] <= 109`
